AI Resume Screener (ATS Simulation System)

Developed an NLP-based resume screening system to compare resumes with job descriptions.

Implemented skill extraction using regex-based matching and a predefined skills database.

Used TF-IDF vectorization and cosine similarity to compute content relevance score.

Designed a weighted scoring model to generate ATS score with improvement suggestions.

Tech Stack: Python, Scikit-learn, Regex, PDFPlumber, NLP
